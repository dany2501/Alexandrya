# Plantillas de Correo (`CT-###`)

Plantillas de los correos transaccionales y de reporte. Cada una define: asunto, cuerpo (texto), variables y notificación asociada (`NT-###`). El diseño HTML de alta fidelidad se produce en la fase de UI; aquí se fija el contenido y la estructura.

| ID | Plantilla | Notificación | Estado |
|----|-----------|--------------|--------|
| [CT-001](CT-001-verificacion-correo.md) | Verificación de correo | NT-001 | ✅ |
| [CT-002](CT-002-aviso-acceso.md) | Aviso de nuevo acceso | NT-002 | ✅ |
| CT-003 | Cambio de contraseña | NT-003 | 🔜 |
| CT-004 | Recuperación de contraseña | NT-004 | 🔜 |
| [CT-005](CT-005-pago-exitoso.md) | Pago exitoso | NT-005 | ✅ |
| [CT-006](CT-006-por-vencer.md) | Suscripción por vencer | NT-006 | ✅ |
| CT-007 | Suscripción vencida | NT-007 | 🔜 |
| CT-008 | Renovación exitosa | NT-008 | 🔜 |
| [CT-009](CT-009-reporte-semanal.md) | Reporte semanal | NT-009 | ✅ |
| CT-010 | Beneficio de referido | NT-010 | 🔜 |
| CT-011 | Contacto recibido (interno) | NT-011 | 🔜 |
| CT-012 | Pago rechazado | NT-012 | 🔜 |
| CT-013 | Anuncio (difusión por correo) | NT-013 | 🔜 |
